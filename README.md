# Smart City Management Platform using IoT

## Overview
This project is an IoT-based Smart City Management Platform developed using ESP32 .

## Features
- Smart street light automation
- Weather monitoring
- Garbage level monitoring
- Real-time sensor data display

## Components Used
- ESP32
- DHT22 Sensor
- LDR Sensor
- HC-SR04 Ultrasonic Sensor
- LED

## Software Used
- Arduino IDE
- Wokwi Simulator

## Working
- LDR controls automatic street lighting
- DHT22 monitors temperature and humidity
- Ultrasonic sensor detects garbage level

## Output
The sensor values are displayed in the Serial Monitor.
